# selectDataMode

selectDataMode option
Type of mode to highlight selected data points with mouse events. (Shift + mouse drag)
@see{selectDataModeDescription}
Default: area
Options: area, transparency, name
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Circular, Correlation, DotLine, Dotplot, Gantt, Genome, Heatmap, Line, Map, Network, ParallelCoordinates, Pie, Sankey, Scatter2D, Scatter3D, ScatterBubble2D, Stacked, StackedLine, StackedPercent, StackedPercentLine, TagCloud, Tree, Treemap, Venn

# selectDataModeDescription

selectDataModeDescription option
Variable label to display when highlighting selected data points that must be in the 'data.z' object. (Shift + mouse drag)
@see{selectDataMode}
Default: false
Options: false
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Circular, Correlation, DotLine, Dotplot, Gantt, Genome, Heatmap, Line, Map, Network, ParallelCoordinates, Pie, Sankey, Scatter2D, Scatter3D, ScatterBubble2D, Stacked, StackedLine, StackedPercent, StackedPercentLine, TagCloud, Tree, Treemap, Venn

# selectedBackgroundColor

selectedBackgroundColor color
The color for selected data points.
Default: rgb(255,0,0)
Used in: Area, AreaLine, Bar, BarLine, Boxplot, Circular, Correlation, DotLine, Dotplot, Gantt, Genome, Heatmap, Line, Map, Network, ParallelCoordinates, Pie, Sankey, Scatter2D, Scatter3D, ScatterBubble2D, Stacked, StackedLine, StackedPercent, StackedPercentLine, TagCloud, Tree, Treemap, Venn

# selectedDataPoints

selectedDataPoints array
Variable names of data points (in the data object) to initially select.
Default: []
